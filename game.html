基本操作

クリックのみで操作可能

キー操作 :

  • Zキー・Enterキー決定・次へ
  • Xキーキャンセル・戻る・メニューを開く
  • 十字キー移動
  • Escキーゲームをやめる

タイトル画面

1 : 各機能

  1. 1-1 : はじめから ゲームを初めから始める
  2. 1-2 : つづきから セーブデータをロードしてゲームを始める
  3. 1-3 : チャプター選択 各チャプターからゲームを始める
  4. 1-4 : バックログ ゲームテキストのログを閲覧する
  5. 1-5 : おまけシナリオ 追加のシナリオを始める
  6. 1-5 : CG集 CGを閲覧する BGMを聴くこともできる

2 : コンフィグ音量の設定・全画面モードの設定を行う

3 : ゲーム終了ゲームを終了する(Escキーでいつでもゲームを終了できる)

キー操作 :

  • Zキー・Enterキー決定
  • 十字キーボタンの選択

シナリオ画面

1 : ゲーム画面画面をクリックすることで次に進む

2 : メニューボタンメニューを開く

キー操作 :

  • Zキー・Enterキー次に進む
  • Xキーメニューを開く

チャプター画面

1 : 各チャプター開始するチャプターを選ぶ

2 : 戻るボタンタイトルに戻る

3・4 : 前へ・次へチャプターのページを変える

キー操作 :

  • Zキー・Enterキー決定・そのチャプターで開始・次へ/前への選択
  • Xキータイトルに戻る
  • 十字キーボタンの選択・チャプターの移動

バックログ画面

1 : 章選択次の章・前の章のチャプターを選択する

2 : チャプター選択次の・前のチャプターを選択する

3 : 戻るボタンタイトルに戻る

4 : バックログ閲覧したメッセージを確認できる　クリックでそのメッセージからゲームを再開する。

キー操作 :

  • Zキー・Enterキー決定・次へ/前への選択・選択したメッセージからゲームを再開
  • Xキータイトルに戻る
  • 十字キーボタンの選択・バックログの移動
